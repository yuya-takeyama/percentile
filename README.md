# percentile

Show percentile of input numbers.

## Installation

```
$ go install github.com/yuya-takeyama/percentile@latest
```

## Usage

```
$ seq 100 | percentile
50%:    50
66%:    66
75%:    75
80%:    80
90%:    90
95%:    95
98%:    98
99%:    99
100%:   100
```
